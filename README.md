# NFTbuilder
A Simple NFT Generation Tool. 一个简单的NFT生成工具. Developed using Unity3D

# 使用说明
>1: 所有图片大小需要一致
>
>2: 支持图片带有alpha通道
>
>3: 会遍历指定目录下的所有图片(同一部位放到同一个目录下)
>
>4: 指定层级,越小越在底层


# 暂未支持
>1: 随机
>
>2: 部位权重
>
>3: 锚点配置
